git# Personal Developer Portfolio 🚀

A modern, minimalist, and fully responsive personal portfolio website to showcase my projects, skills, and academic journey. 

## 🔗 Live Demo
[View Live Portfolio Here](#) *(Update this link once you deploy your site)*

## ✨ Key Features
* **Dark & Light Mode:** Built-in theme toggle with CSS variables.
* **Fully Responsive:** Mobile-first design ensuring perfect layouts across all devices.
* **Smooth Animations:** Custom typing effects and scroll-reveal animations for better UX.
* **Working Contact Form:** Integrated with Google Sheets (via Google Forms) for seamless message delivery without page reloads.
* **Resume Integration:** One-click viewing and downloading capabilities for my latest CV.

## 🛠️ Tech Stack
* **Markup:** HTML5
* **Styling:** CSS3 (Flexbox, CSS Grid, Media Queries)
* **Interactivity:** Vanilla JavaScript (ES6+)
* **Assets:** FontAwesome Icons & Google Fonts (Poppins)
